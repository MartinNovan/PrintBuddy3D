# Security Policy

Thank you for helping to keep this project secure! 

## Supported Versions

From a security perspective, **only the latest version** (or the current state of the `main` branch) is actively maintained and updated. 

If you are using an older version and a bug is discovered in it, a fix will only be released as part of a new update (new release). We therefore recommend keeping the software up to date at all times.

| Version | Receives security fixes |
| ------- | ----------------------- |
| Latest version / `main` | ✅ Yes |
| Any older version       | ❌ No  |

## How to report a security vulnerability

Please **do not report security bugs publicly** (e.g., via public GitHub Issues) to prevent them from being exploited before we can fix them.

If you discover a vulnerability, please choose one of the following methods:

1. **Email us at [martinnovan01@gmail.com](mailto:martinnovan01@gmail.com)**
2. **Use the [GitHub Private Vulnerability Reporting](https://github.com/MartinNovan/PrintBuddy3D/security/advisories)** directly in this repository.

In your report, please include:
* A description of the vulnerability and how it can be exploited.
* Steps to reproduce the issue.
* A proposed fix, if you know how to resolve the issue.

We will try to acknowledge your report within 3 business days and keep you updated on our progress toward a fix.
